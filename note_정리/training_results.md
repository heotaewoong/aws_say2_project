# CheXNet 학습 결과 비교

## 실험 환경

- **모델**: DenseNet-121 (ImageNet pretrained)
- **이미지 크기**: 448×448
- **전처리**: CLAHE
- **인스턴스**: ml.g4dn.16xlarge (Spot)
- **Loss**: BCE + pos_weight (cap=5) + 마스킹(-1 제외)
- **Optimizer**: Adam (lr=1e-4)
- **Scheduler**: ReduceLROnPlateau

## 4차 분석

### 긍정적 변화
- 데이터 20K → 220K로 mAUROC **+0.044** 향상
- Precision 0.2728 → **0.3318** (+0.059) 유의미한 개선
- epoch 1부터 mAUROC 0.76으로 시작 (3차는 0.57에서 시작)

### 문제점 (과적합)
- Best가 **epoch 2**에서 나옴 → 너무 빠른 과적합
- epoch 2 이후 val_loss 지속 상승 (train_loss는 계속 하락)
- 원인: lr=1e-4 과다, Adam 정규화 없음, Gradient clipping 없음

## 5차 — ConvNeXt-L (220K) / 진행 예정

### 변경사항 및 4차 문제 해결

| 항목 | 4차 | **5차** | 변경 이유 |
|------|-----|---------|---------|
| 모델 | DenseNet-121 (8M) | **ConvNeXt-L (198M, 22K)** | 성능 향상 |
| Optimizer | Adam | **AdamW (weight_decay=1e-4)** | 과적합 억제 |
| Scheduler | ReduceLROnPlateau | **Warmup(3ep) + CosineAnnealing** | 안정적 학습 |
| LR | 1e-4 | **3e-5** | epoch 2 과적합 방지 |
| Batch size | 16 | **8** | ConvNeXt-L 메모리 대응 |
| Grad Clipping | 없음 | **max_norm=1.0** | gradient 폭발 방지 |
| Patience | 5 | **7** | 충분한 학습 시간 확보 |
| Epochs | 15 | **20** | 더 많은 학습 허용 |
